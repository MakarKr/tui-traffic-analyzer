# Отказ от ответственности

## Важное предупреждение

Этот инструмент предназначен **исключительно для образовательных целей** и тестирования безопасности **собственных сетей** с разрешения владельцев.

**Запрещается использование в незаконных целях**, включая:
- Перехват трафика без явного согласия владельцев сети
- Атаки на системы без разрешения
- Нарушение законов о защите данных (GDPR, CCPA и др.)
- Любые действия, нарушающие законодательство вашей страны

Авторы не несут ответственности за любое незаконное использование данного инструмента.

## Ограничения HTTPS

Инструмент может анализировать только **метаданные HTTPS-соединений** (SNI, сертификаты), но не может расшифровывать содержимое без:
- Приватного ключа сервера
- Конфигурации клиентского браузера
- Предварительно согласованных ключей

Для полной расшифровки требуется законное разрешение и технические возможности, выходящие за рамки этого инструмента.
